# Bank Account Management System (C++)

## Project Description
This is a simple console-based Bank Account Management System built using C++. It demonstrates Object-Oriented Programming (OOP) concepts and simulates basic banking operations such as creating accounts, depositing money, withdrawing money, and viewing account details.

## Features
- Create a new bank account
- Deposit money into an account
- Withdraw money with balance validation
- View account details
- Menu-driven interface


## Concepts Used
- Classes and Objects (OOP)
- Functions
- Vectors for dynamic storage
- Conditional statements (if-else)
- Loops (do-while)
- Basic data handling
